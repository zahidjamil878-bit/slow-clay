# slow-clay
A single-page pottery studio site with an animated spinning wheel hero, scroll-triggered pot reveal, class schedule, gallery grid, and booking CTA — built in plain HTML/CSS/JS with a warm clay color palette and responsive mobile nav.
Name	Hex	Use
Charcoal	
#2B2622	Text, dark backgrounds
Stoneware	
#DCD3C4	Page background
Slip	
#F5F1E8	Light section background
Glaze	
#5C7A6E	Accent green
Ember	
#C1622A	Primary CTA/accent orange
Running Locally

No build step needed — just open index.html directly in a browser, or serve it with any static file server:

bash
python3 -m http.server
